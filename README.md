# Ec2IpRanges

Check if an IP Address belongs to the Public IP Address Ranges for Amazon EC2

## Installation

    gem 'ec2_ip_ranges'


## Usage

    require 'ec2_ip_ranges'

    Ec2IpRanges.ec2_ip?('67.202.0.0') # => true
